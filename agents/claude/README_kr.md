# fQRGen Claude Code Plugin

> **이 플러그인은 [f-claude-plugins](https://github.com/Finfra/f-claude-plugins) 레포지토리로 이전되었습니다.**

## 새 위치

fQRGen Claude Code 플러그인은 통합 플러그인 레포지토리에서 관리됩니다:

- **레포지토리**: [Finfra/f-claude-plugins](https://github.com/Finfra/f-claude-plugins)
- **경로**: `fQRGen/`

## 설치 방법

```
/plugin marketplace add Finfra/f-claude-plugins
/plugin install fqrgen@f-claude-plugins
```

## 수동 설치

```bash
git clone https://github.com/Finfra/f-claude-plugins.git
cp -r f-claude-plugins/fQRGen/plugin.json .claude-plugin/plugin.json
cp -r f-claude-plugins/fQRGen/skills .claude/skills
```
