---
description: "QR 코드 생성 워크플로우 (fQRGen REST API 사용)"
---

1. **서버 구동 확인**:
   - `http://localhost:3014` (fQRGen REST API) 가 응답하는지 확인합니다.
   - 응답하지 않으면 사용자에게 fQRGen 앱을 실행해달라고 요청합니다.

2. **QR 코드 생성**:
   - `POST /api/generate` 엔드포인트를 호출하여 QR 코드를 생성합니다.
   - 데이터는 사용자가 입력한 URL이나 텍스트를 사용합니다.

3. **결과 확인**:
   - 파일이 성공적으로 생성되었는지 확인하고, 크기 및 포맷을 검증합니다.
   - Mac 환경일 경우 `open` 명령어를 통해 이미지를 띄워줍니다.
