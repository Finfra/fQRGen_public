# fQRGen

> **The moment you copy, your QR code is ready.**

macOS Menu Bar QR Code Generator. Instantly generate QR codes from copied URLs or text, with built-in REST API support.

## Features

- **Instant Access** - Generate QR codes from the menu bar icon
- **Clipboard Automation** - Auto-detects copied URL/text
- **Versatile Formats** - PNG (web) and SVG (print) export
- **QR Code History** - Reuse previously generated codes
- **Configurable Settings** - Multi-language, color, size customization
- **Built-in REST API** - Programmatic QR code generation

## Requirements

- macOS 14.0 or later
